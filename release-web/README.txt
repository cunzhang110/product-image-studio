批量生图大师 - 发布包

这个目录可以直接发给其他人使用。

使用要求
1. 电脑里安装 Node.js
2. 不需要再执行 npm install

Windows 使用方法
1. 双击 start-windows.bat
2. 浏览器打开 http://localhost:3000

Mac 使用方法
1. 终端进入本目录
2. 运行: chmod +x start-mac.command
3. 双击 start-mac.command 或执行 ./start-mac.command
4. 浏览器打开 http://localhost:3000

首次进入页面后，在界面里填写云雾API Key 并保存即可。
